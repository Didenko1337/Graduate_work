# Моделирование колебаний пьезоэлектрической пластины
 
Аналитическое моделирование гармонических колебаний круглой пьезокерамической пластины (юниморф) с визуализацией всех физических полей в реальном времени.

---
## Что делает программа
 
Программа решает задачу о вынужденных гармонических колебаниях круглой пьезокерамической пластины, на торцевые грани которой подаётся переменное напряжение ±V0. Задача решается аналитически в рамках модели плоского напряжённого состояния.

---
## Как скачать и запустить
1. Перейти в релизы
2. Скачать `Piezo Simulator`
3. Разархивировать
4. Запустить `Piezo Simulator.exe`

Установка Python не требуется

**Рассчитываемые поля:**
 
| Вкладка | Что отображается |
|---|---|
| Радиальное смещение u(r) | Профиль смещения вдоль радиуса при заданной частоте |
| АЧХ | Амплитудно-частотная характеристика, резонансные частоты |
| Потенциал φ(r, z) | Электрический потенциал на выбранной высоте z |
| Карта φ(r, z) | Двумерное распределение потенциала по сечению пластины |
| Напряжение σ_rr(r) | Усреднённое радиальное механическое напряжение |
| Форма колебаний | Деформированная форма пластины |
